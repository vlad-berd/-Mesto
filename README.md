# 🧪 Тестовый проект: Mesto

## 📖 Описание проекта

**Mesto** — это интерактивное веб-приложение для работы с пользовательскими карточками (фотографиями).

Пользователь может:

* добавлять новые карточки  
* удалять карточки  
* ставить и убирать лайки  

В рамках проекта было проведено регрессионное тестирование по готовым тест-кейсам, а также ретест исправленных дефектов.

---

## 🎯 Цели тестирования

* Проверка основной функциональности сервиса
* Выполнение регрессионного тестирования по готовым тест-кейсам
* Выявление дефектов и оформление баг-репортов
* Проверка исправлений (ретест)

---

## 🧩 Область тестирования

* Работа с карточками (добавление / удаление)
* Лайки
* UI/UX
* Валидация пользовательских действий

## 🐞 Работа с дефектами

В рамках проекта:
* оформлялись баг-репорты на основе результатов тестирования

* функциональные ошибки (лайки, удаление карточек)
* проблемы пользовательского интерфейса
* некорректная обработка действий пользователя
